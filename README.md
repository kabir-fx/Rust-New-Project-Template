# Rust New Project Template

A minimal template for new Rust projects with CI/CD and dev tooling pre-configured.

## Features

- **Makefile** — `make format`, `make lint`, `make test`, `make run`, `make release`
- **GitHub Actions** — Automated linting, formatting, tests, and releases
- **Devcontainer** — Ready for VS Code / Codespaces

## Usage

```bash
make all    # format, lint, test, run
```
